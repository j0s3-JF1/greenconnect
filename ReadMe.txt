OBJETIVO:

Este arquivo tem com o intuito de auxiliar o usuário em algumas partes do nosso sistema.

========================================================================================================

Fizemos este ReadMe para auxiliar você (Usuário) que irá avaliar nosso projeto, vamos apenas fazer um
pequeno guia para que seja fluido a sua experiência.

Na tela 'LOGIN', fizemos uma lógica onde você precisará apenas preencher os dois campos e apertar no
botão 'LOGIN' para poder acessar a proxima tela do sistema.

Quando efetuar o 'LOGIN' no sistema, você poderá navegar pelas abas que serão disponibilizadas ao 
funcionário da Schneider. Na aba 'GAMES' você poderá clicar também nos Jogos, com isso você será 
direcionado para os respectivos jogos.
